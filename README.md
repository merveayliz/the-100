# 🌍 The 100 Fan Hub

Bu proje, "The 100" hayranlarının bir araya gelebileceği, karakterleri tanıyıp dizinin evreninde gezinebileceği interaktif bir web platformu değil arayüz tasarımıdır. ark os web bu sitenin güncellenmiş web platformudur. 
*Not: Bu site şu an için sadece masaüstü (PC) görünümü için optimize edilmiştir.*

---

## 🛠 Özellikler | Features

### 🇹🇷 Türkçe Özellikler
* **Dinamik Giriş Deneyimi:** Başlığın yanında dönen dünya animasyonu ve eşzamanlı olarak rengi değişen, kendini güncelleyen alt metinler.
* **Kullanıcı Hafızası:** JavaScript (localStorage) kullanılarak giriş formuna yazılan isim sistem tarafından hatırlanır.
* **İnteraktif Karakter Kartları:** Tüm karakter listesi JavaScript ile dinamik olarak ekrana basılır. Kartlara tıklandığında arka plan bulanıklaşarak karakter detayları açılır.
* **Gelişmiş Filtreleme:** "İlginç Bilgiler" bölümünde kullanıcıların arama yapabileceği ve kategorilere göre filtreleme yapabileceği bir sistem mevcuttur.
* **Modern UI/UX:** Bolca hover efekti, şık navigasyon menüsü ve kullanıcı dostu animasyonlar.

### 🇺🇸 English Features
* **Dynamic Landing Page:** Features a rotating world animation next to the title and synchronized color-shifting subtitles.
* **Personalized Experience:** Utilizes JavaScript to remember the user's name entered in the login form.
* **Interactive Character Directory:** Character cards are dynamically rendered via JS. Clicking a card triggers a blur effect on the background and displays detailed info.
* **Search & Filter System:** An advanced search bar and filtering mechanism in the "Fun Facts" section.
* **Modern UI/UX:** Rich hover effects, a sleek navigation menu, and smooth animations.

## 💻 Kullanılan Teknolojiler | Tech Stack

* **HTML5** - Yapısal kurgu ve navigasyon.
* **CSS3** - Gelişmiş animasyonlar, hover efektleri ve layout.
* **JavaScript (ES6+)** - Dinamik karakter kartları, isim hatırlama fonksiyonu, filtreleme algoritması ve DOM manipülasyonu.
